## 简介
fl-basic-package管理项目中常用的依赖库，锁定特定版本，保证所有项目中使用的版本一致性

## 管理基础库列表

|依赖库  |版本号   |
|----------------|-------------------|
|react|16.8.6|
|react-dom|16.8.6|
|react-router-dom|4.3.1|
|prop-types|15.7.2|
|dva|2.4.1|
|dva-loading|2.0.1|
|antd|3.26.13|
|axios|0.18.0|

## 使用说明

### 1.全局安装
npm install fl-basic-package -g;

### 2.项目根目录运行命令
### 1.) fl-package // 先删除依赖库，再安装
### 2.) fl-package uninstall // 删除依赖库
### 3.) fl-package -v // 工具包版本
